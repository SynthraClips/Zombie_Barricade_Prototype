# Zombie Barricade Prototype

Godot 4.7 prototype focused on surviving a moving zombie route with a squad, barricades, upgrades, and hero call-ins.

## Controls

- Move / aim: Mouse
- Fire: Auto-fire, or hold left mouse when auto-fire is disabled
- Deploy Barricade: **B** or the HUD button
- Call Hero: **H** or the HUD button
- Hero Ultimate: **U** or the HUD button (while the hero is active)
- Pause: **Escape**

The HUD buttons show when an action is locked, active, or cooling down. A hero must be selected before a run to use hero call-in controls.
